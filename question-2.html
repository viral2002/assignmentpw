html entities

HTML entities are special codes used to represent characters that cannot be easily typed or displayed using regular keyboard input. They are commonly used in HTML documents to display reserved characters, symbols, special characters, and characters with special meanings in HTML syntax.

  1. < represents the less-than symbol (<).
  2. > represents the greater-than symbol (>).
  3. & represents the ampersand symbol (&) itself (used to escape the & symbol).
  4. " represents the double quotation mark (").
  5. © represents the copyright symbol (Â©).
