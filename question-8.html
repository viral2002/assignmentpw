what is html

HTML stands for Hypertext Markup Language. It is the standard markup language used for creating web pages and applications. HTML uses a set of tags to structure the content and define the layout and appearance of web pages. HTML documents are made up of elements, which are represented by tags enclosed in angle brackets (< >). The tags define the purpose and structure of different parts of a web page, such as headings, paragraphs, images, links, lists, and more.

what is semanitc html

Semantic HTML refers to the practice of using HTML elements that convey meaning and structure to web content. It involves choosing the appropriate HTML tags that best represent the purpose and meaning of the content, rather than relying on generic or presentational tags. By using semantic HTML, developers provide additional information about the content's meaning, making it more accessible to assistive technologies and search engines. It also helps improve code readability, maintainability, and overall understanding of the document structure.

list of commnaly used semanitc tag

  • <header>: Represents the introductory content or container for the introductory content of a section or page.
  • <nav>: Defines a section containing navigation links.
  • <main>: Represents the main content of a document.
  • <article>: Defines a self-contained, independent piece of content, such as a blog post or news article.
  • <section>: Represents a thematic grouping of content within a document.
